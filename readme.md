# MovieHD
This is a clean architecture app  built with

- Coroutines
- MVVM
- Extension Functions
- Dagger Hilt
- Retrofit
- Room
- Navigation Components
- Cache Strategy (Repository Pattern)
- Coroutines Flow
- ViewBinding
- Livedata
- Material design theming
- Support for dark and light mode
